public enum CompressionMode : uint

Compression mode

Values

none

No compression.

text

Compress the contents stream of the page.

image

Compress the streams of the image objects.

metadata

Other stream datas (fonts, cmaps and so on) are compressed.

all

All stream datas are compressed
