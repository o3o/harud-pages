public HPDF_STATUS useJPEncodings()

Enables Japanese encodings.

After useJPEncodings() is invoked, an application can use the following Japanese encodings:

  • 90ms-RKSJ-H
  • 90ms-RKSJ-V
  • 90msp-RKSJ-H
  • EUC-H
  • EUC-V
