public HPDF_STATUS useJPFonts()

Enables Japanese fonts. After useJPFonts() is involed, an application can use the following Japanese fonts

  • MS-Mincyo
  • MS-Mincyo,Bold
  • MS-Mincyo,Italic
  • MS-Mincyo,BoldItalic
  • MS-Gothic
  • MS-Gothic,Bold
  • MS-Gothic,Italic
  • MS-Gothic,BoldItalic
  • MS-PMincyo
  • MS-PMincyo,Bold
  • MS-PMincyo,Italic
  • MS-PMincyo,BoldItalic
  • MS-PGothic
  • MS-PGothic,Bold
  • MS-PGothic,Italic
  • MS-PGothic,BoldItalic
