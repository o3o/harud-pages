public HPDF_STATUS setSize(
    PageSizes size, 
    PageDirection direction)

Changes the size and direction of a page to a predefined size

Parameters

size

Specify a predefined page-size value. The following values are available:

  • PageSizes.LETTER
  • PageSizes.LEGAL
  • PageSizes.A3
  • PageSizes.A4
  • PageSizes.A5
  • PageSizes.B4
  • PageSizes.B5
  • PageSizes.EXECUTIVE
  • PageSizes.US4x6
  • PageSizes.US4x8
  • PageSizes.US5x7
  • PageSizes.COMM10

    • direction

    specify the direction of the page

  • PageDirection.PORTRAIT
  • PageDirection.LANDSCAPE
