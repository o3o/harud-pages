public HPDF_STATUS setSlideShow(
    HaruTransitionStyle type, 
    float disp_time, 
    float trans_time)

Configures the setting for slide transition of the page.

Parameters

type

The transition style. The following values are available.

  • HaruTransitionStyle.WIPERIGHT</li> <li>HaruTransitionStyle.WIPEUP
  • HaruTransitionStyle.WIPELEFT</li> <li>HaruTransitionStyle.WIPEDOWN
  • HaruTransitionStyle.BARNDOORSHORIZONTALOUT</li> <li>HaruTransitionStyle.BARNDOORSHORIZONTALIN
  • HaruTransitionStyle.BARNDOORSVERTICALOUT</li> <li>HaruTransitionStyle.BARNDOORSVERTICALIN
  • HaruTransitionStyle.BOXOUT</li> <li>HaruTransitionStyle.BOXIN
  • HaruTransitionStyle.BLINDSHORIZONTAL</li> <li>HaruTransitionStyle.BLINDSVERTICAL
  • HaruTransitionStyle.DISSOLVE
  • HaruTransitionStyle.GLITTERRIGHT</li> <li>HaruTransitionStyle.GLITTERDOWN
  • HaruTransitionStyle.GLITTERTOPLEFTTOBOTTOM_RIGHT
  • HaruTransitionStyle.REPLACE

    • disp_time

    The display duration of the page. (in seconds)

    trans_time

    The duration of the transition effect. Default value is 1(second).
